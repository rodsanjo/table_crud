Class TCPDF_Adapter

TCPDF PDF Rendering interface

TCPDF_Adapter provides a simple, stateless interface to TCPDF.

Unless otherwise mentioned, all dimensions are in points (1/72 in). The coordinate origin is in the top left corner and y values increase downwards.

See http://tcpdf.sourceforge.net for more information on the underlying TCPDF class.

TCPDF_Adapter implements Canvas
