estudiar_query_string()

estudiar_query_string()

Realiza el estudio del requerimiento http recibido y elige el controlador y método que se ejecutará para atenderla.

Solo debe invocarse desde la clase aplicación. Para ejecutar un controlador desde otro controlador (forwarding) debe usarse el método cargar_controlador() de esta clase.
